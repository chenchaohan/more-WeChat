# Getting started
下载包后，
* mac 直接打开 more-wechat.command
    * 打开后弹窗警告点“好“
    * 在窗口填入数字或者直接enter
* 或者打开终端 
    * 搜索 终端后打开
    * 执行 `chmod +x more-wechat.sh`
    * 执行 `./more-wechat.sh`
